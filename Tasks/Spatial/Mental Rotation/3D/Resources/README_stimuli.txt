There are three folders of stimuli. Each folder contains the identical stimuli in different formats and naming conventions.


-A folder with All 384 stimuli as JPG, named 1-48 with each 4 angular disparities (0, 50, 100, 150) and two versions ('Same' trials and 'Different' trials, appended with R)

-A folder with all stimuli as .pct. This filetype is preferred by Psyscope software for Mac, which was used to program this experiment
(Cohen, J. D., MacWhinney, B., Flatt, M., & Provost, J. (1993). PsyScope: A new graphic interactive environment for designing psychology experiments. Behavioral Research Methods, Instruments, and Computers, 25(2), 257-271.)

-Table 2 in the paper reports the correspondence between the extendede and descriptive names for the subset of stimuli used in the study.
